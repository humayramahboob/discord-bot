# Anime Tracker Discord Bot

This is a personal Discord bot project I created to use with my friends, who enjoy watching anime and like to keep track of what they're watching. It helps us organize our anime lists, track progress, and discover seasonal shows all within Discord.

---

## Features

- **Track Anime Progress:**  
  Add anime to your personal watchlist, update the episode you're on, and mark shows as watching, watched, or want to watch.

- **View Tracked Lists:**  
  See your anime watchlist filtered by status (watching, watched, want to watch) with pagination.

- **Seasonal Anime Browsing:**  
  Browse popular anime airing in the current or selected season with a paginated list.

- **Interactive UI:**  
  - Status or season dropdown filters.  
  - Pagination controls to navigate through lists.  
  - Zoom button to toggle between list view and a large cover image preview for each anime.

- **Notifications:**  
  Receive direct messages when new episodes air for anime you're tracking.

- **Aliases Support:**  
  Use short aliases to quickly refer to anime in commands.

- **Multi-user Support:**  
  Track and view lists for yourself or your friends.

- **Timezone Awareness:**  
  Airing times are shown in your configured timezone.
